# 音乐下载工具
主流网站音乐下载

# 支持:
- 网易云
- QQ音乐
- 酷狗音乐
- 千千静音
- 虾米音乐
- 酷狗音乐

# 使用
###开打Music目录 

###运行 Demo.py 后者 Cmd.py:

Demo.py 界面是交互界面需要安装tkinter
Cmd.py 界面是代码界面

# 平台
Windows  
Python3.6
